    There is no single cloud that delivers every business function!.

    Enterprises are moving toward consumption of IT on demand, using a mix of public cloud, private cloud and multiple software-as-a-service (SaaS) vendors, as well as company-owned resources. Managing multiple vendors of cloud services, then, is already becoming the norm. Although IaaS providers like AWS and SaaS providers like Salesforce have set a high bar, there is no single cloud that delivers every business function.

    The result is that the enterprise cloud strategy is more complex than 'in cloud' or 'not in cloud' -it already encompasses multiple cloud environments. Many organisations are now realising thecosts and complexity of managing them and the longterm unsustainable approach.
